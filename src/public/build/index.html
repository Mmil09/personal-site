Skills & Services

I am a professional software engineer working in a small, fast-paced startup environment, working mainly in JavaScript, Node, and Front End Web Development. I am excited about the direction of the modern web and the growing ecosystem of JavaScript libraries and frameworks. Alongside web development, I have been developing projects for Arduino using C/C++.

  • Javascript and jQuery
  • HTML and Templating languages (Jade, Handlebars)
  • CSS and CSS preprocessors (Stylus, SASS, LESS)
  • Node/Express
  • API development
  • Build systems (Gulp and Grunt)
  • Integration of 3rd party services
  • Angular
  • React
  • React Native
